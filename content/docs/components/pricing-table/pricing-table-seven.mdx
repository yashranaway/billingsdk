## Usage

```tsx
import { PricingTableSeven } from "@/components/billingsdk/pricing-table-seven";
import { plans } from "@/lib/billingsdk-config";

<PricingTableSeven
  plans={plans}
  title="The Perfect Plan For Your Needs"
  description="Our transparent pricing makes it easy to find a plan that works within your financial constraints."
  onPlanSelect={(planId) => console.log('Selected plan:', planId)}
/>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `plans` | `PlanProps[]` | Array of pricing plans |
| `onPlanSelect` | `(planId: string) => void` | Callback when a plan is selected |


## Theming

The pricing table component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
