## Usage

```tsx
import { CancelSubscriptionCardTwo } from "@/components/billingsdk/cancel-subscription-card-two";
import { plans } from "@/lib/billingsdk-config";
```
```tsx
<CancelSubscriptionCardTwo
  title="We're sorry to see you go..."
  description="Before you cancel, let us know what we could do better."
  plan={plans[1]}
  onCancel={async (planId) => {
    // Handle cancellation
    await cancelSubscription(planId);
  }}
  onKeepSubscription={async (planId) => {
    // Handle keeping subscription
    await keepSubscription(planId);
  }}
/>
```

## Props

| Prop                 | Type                                             | Required | Description                                |
|----------------------|--------------------------------------------------|----------|--------------------------------------------|
| `title`              | `string`                                        | ✅       | Main card title                            |
| `description`        | `string`                                        | ✅       | Card description text                       |
| `plan`               | `Plan`                                          | ✅       | Plan object with subscription info          |
| `onCancel`           | `(planId: string) => Promise<void> \| void`     | ✅       | Callback for cancel action                  |
| `warningText`        | `string`                                        | ❌       | Warning message text                        |
| `supportText`        | `string`                                        | ❌       | Title for customer support                  |
| `supportLink`        | `string`                                        | ❌       | Link for customer support          |
| `keepButtonText`     | `string`                                        | ❌       | Text for keep subscription button           |
| `continueButtonText` | `string`                                        | ❌       | Text for continue button                    |
| `finalTitle`         | `string`                                        | ❌       | Final confirmation screen title             |
| `finalSubtitle`      | `string`                                        | ❌       | Final confirmation screen subtitle          |
| `goBackButtonText`   | `string`                                        | ❌       | Text for go-back button                     |
| `confirmButtonText`  | `string`                                        | ❌       | Text for final confirm button               |
| `onKeepSubscription` | `(planId: string) => Promise<void> \| void`     | ❌       | Callback for keep subscription action       |
| `className`          | `string`                                        | ❌       | Additional CSS classes                      |


## Theming

The cancel subscription card component is styled using the `shadcn/ui` library. You can customize the colors and fonts by overriding the CSS variables. You can also get the theme from the [Theming](/docs/theming) page.
